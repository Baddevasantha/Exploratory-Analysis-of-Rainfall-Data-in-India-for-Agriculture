Overview

Exploratory Analysis of Rainfall Data in India for Agriculture is a Machine Learning web 
application that predicts whether it will rain tomorrow based on weather parameters.
The system helps farmers and agricultural planners make better irrigation and crop
decisions using real-time predictions. The best-performing model (Random Forest)
achieved 85.69% accuracy on the test dataset.
